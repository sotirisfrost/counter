# June 7 ✦

A minimal countdown page to June 7th. Built to live on GitHub Pages and shared with friends.

## What it is

A single-file HTML countdown timer. Clean, typographic, no dependencies beyond two Google Fonts. It ticks live every second and shows days, hours, minutes, and seconds remaining until June 7th midnight.

When the day arrives, the counter quietly becomes: *He arrived!*

```
https://sotirisfrost.github.io/counter/
```

## Notes

- If June 7th of the current year has already passed, the counter automatically targets the following year
- No build step, no dependencies, no tracking — just a file
- Works on mobile
